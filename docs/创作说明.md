# 创作说明

## 创作背景

本项目源于与学校美术老师的深入交流。在与老师商讨业务流程和用户故事的过程中，了解到当前美术教学面临的痛点：学生创作后缺乏即时、个性化的反馈指导，教师精力有限难以逐一辅导每位学生。基于这一真实教学需求，本人设计并开发了"晋彩智绘"平台，将 AI 导师引入美术创作教学场景，让每位学生都能获得即时的多维度评审与对话式辅导。

## 原创与参考情况

本作品为本人独立原创开发，非团队作品。前端采用 Vue 3 + TypeScript + Vite + Pinia 构建，后端采用 Spring Boot 3 + Java + JPA，并通过自部署的 New API 统一网关对接上游大模型服务。产品设计、交互流程、UI、前后端业务代码及网关部署均由本人独立完成。

## AI 工具使用情况

1. **AI 导师（核心功能）**：经统一网关调用国产多模态大语言模型，对学生生成的图像从光影构图、艺术风格、内容完整性等维度进行评审打分，并通过 SSE 流式输出提供对话式辅导，呈现于学生工作台的"AI 导师"抽屉面板中。该功能直接回应了美术老师"希望学生能获得即时反馈"的核心诉求。

2. **图像生成**：经统一网关调用国产图像生成模型，用于将学生提示词转化为图像作品，呈现于学生工作台与作业提交页。

3. **提示词优化**：调用国产大语言模型，对学生输入的中文提示词进行结构化改写与扩写，辅助学生更准确地表达创意，呈现于"提示词优化"组件中。

所有 AI 模型仅作为辅助创作与教学的工具嵌入产品功能中，不参与项目本身的代码、设计与文档创作。
