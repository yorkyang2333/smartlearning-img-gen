# 智绘 (SmartCanvas) — 测试说明

## 一、环境准备

| 依赖 | 版本要求 | 说明 |
|------|---------|------|
| Java JDK | 17+ | 推荐 Eclipse Temurin 17 |
| Node.js | 20+ | 前端构建与开发服务器 |
| Maven | 3.8+ | 后端构建（项目含 mvnw wrapper） |
| MySQL | 8.0+ | 业务数据库 |
| Docker | 20+ | 运行 AI Gateway 容器 |

### 数据库初始化

```bash
mysql -u root -e "CREATE DATABASE IF NOT EXISTS smartcanvas CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
```

首次启动后端时，Hibernate 会自动建表（`ddl-auto: update`），DataSeeder 会自动写入测试账号和默认模型。

## 二、一键启动

```bash
sh start.sh
```

启动顺序：AI Gateway (4000) → Spring Boot 后端 (8080) → Vue 前端 (5173)。

按 `Ctrl+C` 可安全关闭所有服务。

## 七、常见问题

| 问题 | 解决方案 |
|------|---------|
| 后端启动报数据库连接失败 | 确认 MySQL 已启动，且存在 `smartcanvas` 数据库 |
| 网关容器启动失败 | 确认 Docker 已运行（`docker ps`） |
| 前端 API 请求 CORS 报错 | 确认后端运行在 8080 端口，前端在 5173 端口 |
| 模型同步为空 | 先在网关面板中添加至少一个上游渠道 |
