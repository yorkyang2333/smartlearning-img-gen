# 晋彩智绘 — 用户手册

## 一、产品简介

晋彩智绘是一款专为美术教学场景设计的 AI 图像生成平台。平台采用教师-学生双角色架构：教师管理 AI 模型、布置创作作业、监控学生活动并批改作品；学生通过对话式界面进行 AI 图像创作，并可获得 AI 学伴的即时辅导。

## 二、教师端使用指南

### 2.1 登录

访问 http://localhost:5173，使用教师账号登录（默认：teacher / 123456）。登录后自动跳转到教师仪表盘。

### 2.2 仪表盘 (Dashboard)

教师首页，展示：
- 学生总数与活跃情况
- 近期生成统计（每日/每周）
- 作业完成进度概览

### 2.3 模型目录 (Models)

管理平台可用的 AI 模型：

1. **同步网关模型**：点击"同步 AI Gateway 模型"，自动从网关拉取可用模型列表
2. **模型配置**：为每个模型设置展示名称、描述、支持的生成尺寸
3. **模型启停**：控制哪些模型对学生可见
4. **批量操作**：支持批量启用/禁用模型
5. **网关管理**：配置 AI Gateway 的连接地址和 API Key
6. **渠道查看**：查看网关中已配置的上游渠道状态

### 2.4 学生管理 (Students)

- 查看所有学生列表
- 添加单个学生账号
- 批量添加学生
- 启用/禁用学生账号
- 设置学生生成配额
- 重置学生密码
- 删除学生

### 2.5 作业管理 (Assignments)

- **创建作业**：设置标题、描述、参考图片、截止日期
- **查看提交**：浏览学生提交的作品
- **批改评分**：为学生作品打分并给出文字反馈

### 2.6 模板管理 (Templates)

创建和管理提示词模板，供学生在创作时参考使用。

### 2.7 实时监控 (Live)

实时查看学生当前的生成活动流，了解课堂进度。

### 2.8 设置 (Settings)

- 配置 AI 学伴：选择对话模型、编辑系统提示词
- 默认学伴模型为 `deepseek-v4-flash`，可切换为任意已同步的文本模型

## 三、学生端使用指南

### 3.1 登录

访问 http://localhost:5173，使用学生账号登录（默认：student / 123456）。登录后自动跳转到创作工作区。

### 3.2 创作工作区 (Workspace)

核心创作界面，采用对话式交互：

1. **选择模型**：从教师开放的模型中选择（支持文生图和图生图模型）
2. **输入提示词**：描述想要生成的图像内容
3. **提示词工具**：
   - **提示词构建器 (PromptBuilder)**：通过结构化表单辅助编写提示词
   - **提示词优化 (PromptOptimizer)**：AI 自动优化提示词质量
   - **提示词助手 (PromptHelper)**：提供编写建议
4. **生成图像**：点击发送，等待 AI 生成结果
5. **图生图**：上传参考图片，基于参考进行二次创作
6. **对话管理**：可创建多个对话，切换和删除历史对话

### 3.3 AI 学伴 (Tutor)

点击界面的 AI 学伴按钮，打开 TutorDrawer 对话抽屉：
- 向 AI 导师提问创作技巧
- 获取提示词编写建议
- 了解艺术风格和构图知识
- 支持 SSE 流式输出，实时显示回复

### 3.4 AI 作品评审 (TutorReview)

生成图像后可请求 AI 评审：
- 从光影构图、艺术风格、内容完整性等维度评分
- 提供改进建议

### 3.5 班级画廊 (Class Gallery)

浏览本班同学的生成作品，获取创作灵感，支持点赞互动。

### 3.6 我的作业 (Assignments)

- 查看教师布置的作业列表
- 进入作业详情查看要求
- 进入 AssignmentPlay 按要求创作
- 提交作品等待教师评分

## 四、AI Gateway 管理

### 4.1 访问网关面板

访问 http://localhost:4000，使用管理员账号登录（root / 12345678）。

### 4.2 添加上游渠道

1. 进入"渠道"页面
2. 点击"添加新渠道"
3. 选择渠道类型（DeepSeek、智谱、通义千问等）
4. 填入对应的 API Key
5. 选择该渠道支持的模型
6. 保存并测试连接

### 4.3 默认模型配置

DataSeeder 首次启动时自动写入以下模型：

| 模型 ID | 类型 | 用途 |
|---------|------|------|
| `deepseek-v4-flash` | TEXT_GENERATION | AI 学伴对话（默认） |
| 文生图模型 | TEXT_TO_IMAGE / BOTH | 文生图 / 图生图 |
| 多模态文本模型 | TEXT_GENERATION | 多模态分析 / AI 学伴 |

实际模型 ID 以网关中配置的渠道为准，教师可通过"同步 AI Gateway 模型"拉取最新列表。
