# 晋彩智绘 — 用户手册

## 一、产品简介

晋彩智绘是一款专为教育场景设计的 AI 图像生成平台。平台采用教师-学生双角色架构，教师可管理 AI 模型、布置创作作业、监控学生活动；学生可通过对话式界面进行 AI 图像创作、获得 AI 学伴指导。

## 二、教师端使用指南

### 2.1 登录

访问 http://localhost:5173，使用教师账号登录（默认：teacher / 123456）。

### 2.2 仪表盘 (Dashboard)

登录后进入教师仪表盘，可查看：
- 学生总数与活跃情况
- 近期生成统计
- 作业完成进度概览

### 2.3 模型目录 (Models)

管理平台可用的 AI 模型：

1. **同步网关模型**：点击"同步 AI Gateway 模型"，自动从网关拉取可用模型列表
2. **模型配置**：为每个模型设置展示名称、描述、支持的生成尺寸
3. **模型启停**：控制哪些模型对学生可见
4. **网关管理**：配置 AI Gateway 的连接地址和 API Key

### 2.4 学生管理 (Students)

- 查看所有学生列表
- 添加新学生账号
- 启用/禁用学生账号

### 2.5 作业管理 (Assignments)

- **创建作业**：设置标题、描述、参考图片、截止日期
- **查看提交**：浏览学生提交的作品
- **批改评分**：为学生作品打分并给出文字反馈

### 2.6 实时监控 (Live)

实时查看学生当前的生成活动，了解课堂进度。

### 2.7 模板管理 (Templates)

创建和管理提示词模板，供学生在创作时参考使用。

### 2.8 设置 (Settings)

- 修改个人显示名称和密码
- 配置 AI 学伴：选择对话模型、编辑系统提示词

## 三、学生端使用指南

### 3.1 登录

访问 http://localhost:5173，使用学生账号登录（默认：student / 123456）。

### 3.2 创作工作区 (Workspace)

核心创作界面，采用对话式交互：

1. **选择模型**：从教师开放的模型中选择
2. **输入提示词**：描述想要生成的图像内容
3. **提示词工具**：
   - 提示词构建器：通过结构化表单辅助编写提示词
   - 提示词优化：AI 自动优化提示词质量
4. **生成图像**：点击发送，等待 AI 生成结果
5. **图生图**：上传参考图片，基于参考进行创作

### 3.3 AI 学伴 (Tutor)

点击界面右侧的 AI 学伴按钮，打开对话抽屉：
- 向 AI 导师提问创作技巧
- 获取提示词编写建议
- 了解艺术风格和构图知识

### 3.4 班级画廊 (Class Gallery)

浏览本班同学的生成作品，获取创作灵感。

### 3.5 我的作业 (Assignments)

- 查看教师布置的作业列表
- 进入作业详情，按要求进行创作
- 提交作品等待教师评分

## 四、AI Gateway 管理

### 4.1 访问网关面板

访问 http://localhost:4000，使用管理员账号登录（root / 12345678）。

### 4.2 添加上游渠道

1. 进入"渠道"页面
2. 点击"添加新渠道"
3. 选择渠道类型（OpenAI、Anthropic、Google 等）
4. 填入对应的 API Key
5. 选择该渠道支持的模型
6. 保存并测试连接

### 4.3 推荐模型配置

| 模型 ID | 用途 | 推荐渠道 |
|---------|------|---------|
| gpt-image-1 | 文生图/图生图 | OpenAI |
| dall-e-3 | 文生图 | OpenAI |
| gpt-4o | AI 学伴对话/多模态分析 | OpenAI |
| claude-3-5-sonnet-latest | AI 学伴对话 | Anthropic |
| gemini/gemini-2.5-flash | 文生图 | Google |
